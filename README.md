# LOK Extension

Extension Firefox pour télécharger des vidéos YouTube (clips ou vidéo complète) et copier des médias (images/vidéos) depuis n'importe quel site vers Premiere Pro ou tout autre logiciel.

## Fonctionnalités

- **⬇ Téléchargement clip** — définir un début et une fin, télécharger la portion
- **⬇ Téléchargement complet** — vidéo entière en MP4 (H.264 + AAC)
- **📋 Copier un média** — clic droit sur n'importe quelle image ou vidéo → presse-papier → Ctrl+V dans Premiere Pro
- **Barre de progression** en temps réel dans le popup
- **Téléchargements parallèles** réglables (1–16 connexions)
- Compatible **1000+ sites** via yt-dlp (YouTube, Instagram, Twitter/X, TikTok, Vimeo…)
